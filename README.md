# Source SDK 2005
Custom fork of the 2006 version of the Source SDK to add BSP support for the Original XBox port of Half-Life 2. Intended to recreate the SDK from 2005.

**Pull this repository by using this in the command bar:** ``git clone https://github.com/smb123w64gb/source-sdk-2005-xbox``
